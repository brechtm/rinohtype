FloatStyle¶

class rinoh.flowable.FloatStyle(base=None, **attributes)¶

Style class for Float

float¶

Float the flowable to the top or bottom of the page

Accepts Bool: true or false

Default: false

width¶

Width to render the flowable at (inherited from FlowableStyle)

Accepts FlowableWidth: auto, fill or a numeric value followed by a unit (pt, in, pc, mm, cm, %, /4)

Default: auto
