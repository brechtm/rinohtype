    Strings (rinoh.strings)¶

    class rinoh.strings.Strings¶

    Stores several StringCollections

    class rinoh.strings.StringField(key, style=None, parent=None, source=None, user=False)¶

    Styled text that will be substituted with a configured string

    The displayed string is either the localized string as determined by the language set for the document or the user-supplied string passed to the TemplateConfiguration.
